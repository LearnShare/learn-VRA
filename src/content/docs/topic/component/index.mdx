组件是可复用的模块。它们类似于 HTML 元素，一般包含了下面几个部分：

+ 模板: 包含 HTML 元素或自定义组件，最终渲染为 HTML
+ 样式: 最终编译为 CSS，这部分是可选的
+ 数据和逻辑: 处理组件数据和交互逻辑的代码

关于组件：

+ 组件是应用的基本组成单位。
+ 组件内可以包含其他组件，构成组件树。
+ 最先渲染到页面中的组件称为**根组件**。
