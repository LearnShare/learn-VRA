# Vue、React 和 Angular 一起学怎么样？

Why not.

## 目标

- 同时学习 Vue、React 和 Angular
- 同步学习三者相同/相关的知识

- 尽量涉及更多、更广的内容

1. 将学习内容记录到文档中（编写为 Markdown 存放在本项目中）

4. 按日总结并更新
